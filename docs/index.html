Syllabus

Tired of finding errors in your spreadsheets that perpetuate and cascade through your work? Still trying to cram “big” data into tiny worksheets? Wish you had an easier way to easily share data and results with others? Want to better manage and maintain your workflow? Always wanted to learn R and version control but never found the time? If any of your answers are yes, then this course is for you! Building on other successful data science training resources, this course exposes students in transportation research and practice and best practices in scientific computing through hands-on lab sessions.

Format

Classes will all be hands-on sessions with lecture, discussions, and labs. Readings drawn from books, articles, and online resources will be assigned. Students are expected to read them before class and to participate in class discussions. Highlights of the class include creating a work product in which students go through the process of data retrieval, processing, conducting analysis, and developing a report/article while learning the best practices of data science.

Software and Hardware

Participants are responsible for bringing their own laptop with the required software installed. If you’re using a government issued laptop, you might need to reach out to your IT department as some of these installations might required admin rights. The following is a list of required software:

  • R
  • RStudio
  • Git
  • JKD 21 (Windows users: suggest “Install just for you” install option)

If you need additional help installing the required software refer to this setup overview.
